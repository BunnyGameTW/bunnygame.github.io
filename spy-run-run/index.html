【心得】

第一次一個人去挑戰參加game jam就遇到整組都擅長程式的情況，於是我挑戰負責2D美術部分
努力的畫好第一張封面時，才發現原來我腦袋想的畫面(忍者要從惡魔城逃出去)跟大家(特務要從摩天大樓逃出去)完全不同！之後和大家確實溝通好想要呈現的方式後，畫了草圖，確認沒問題後才開始著手，不過最後時間不夠還是有些畫面還是不如預期🫣
下次如果要單人挑戰的話要多練習各種技能，以備不時之需

Present Memories

【遊戲介紹】

在這款視覺小說冒險遊戲中，你將扮演Vox本人去探索他的記憶，想盡一切辦法與他的過去的氏族一起擁有一段難忘的經歷
玩家不但可以欣賞到受浮世繪藝術啓發的精緻CG，還可以獲得跟著名《逆轉裁判》系列相似的成熟遊戲體驗，以及簡單但迷人，充斥著神秘、轉折和許多驚喜的故事情節

【心得】

Vtuber也就是所謂的虛擬偶像，基本上跟現在常見的實況主差不多，只是他是用虛擬角色的形象來進行活動，常見的內容有唱歌、聊天、玩遊戲等等
而這次做的遊戲就是為了慶祝Nijisanji這家企業所屬的英語系Vtuber - Vox Akuma的生日而有的計畫

Vox的聲音真的超低超性感，第一次聽到就被圈粉了，他有時候也超可愛而且很認真直播，周休零日你敢信，快去支持他！
看著他直播玩著自己做的遊戲時好像在作夢，超興奮又好感動🤩
遊戲劇情加了許多直播中發生過的梗，也有刺激打鬥、感動到快哭的情節，重點是音樂超搭、而且CG超漂亮，為什麼大家都這麼才華洋溢！
團隊還做了特設網站，而且繪師Roku還幫所有開發者都畫了人物形象，超有心，看看我多可愛！

第一次用Ren’Py這個專門做視覺小說的遊戲引擎，超著名的心跳文學部也是用這款引擎做的
除了文本邏輯外還負責了畫廊和差不多功能的繪馬，Ren’Py語法非常簡單，卻也能用Python做出很多特別的效果，程式面上基本沒什麼大難關
反而我覺得最困難的地方是在跟外國人溝通，其實這是我第一次跟外國夥伴合作，緊張到沒辦法好好說話，到最後都是用打字回應他們的問題，每當開會時我都感覺在做大型英聽測驗🫠
不過大家都超級善良，一直說我很棒，而且每個團隊都超有效率、氣氛超歡樂，最後的成品我超滿意，真的很開心能一起合作！
